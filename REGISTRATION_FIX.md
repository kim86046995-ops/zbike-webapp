## 🔄 권장 워크플로우

### 신규 등록 시 체크리스트
1. ✅ 업체 목록에서 **중복 사업자번호 확인**
2. ✅ 중복이면 **기존 업체 삭제** 또는 **새 사업자번호 사용**
3. ✅ 모든 필수 항목 입력
   - 상호명
   - 사업자번호 (형식: XXX-XX-XXXXX)
   - 대표자명
   - 대표자 전화번호
   - 대표자 주민번호
   - **주소 검색 버튼 클릭 필수** ⚠️
   - 신분증 사진 업로드 필수 ⚠️
4. ✅ "등록하기" 클릭
5. ✅ 성공 메시지 확인
6. ✅ 자동으로 업체 목록으로 이동

### 중복 방지 팁
- 등록 전에 **업체 목록에서 검색**으로 중복 확인
- 사업자번호는 **정확하게** 입력 (하이픈 자동 생성됨)
- 테스트용은 **111-11-11111** 같은 패턴 사용

## 📊 에러 코드 가이드

### HTTP 400: 잘못된 요청
**메시지**: "이미 등록된 사업자번호입니다."
**원인**: 동일한 사업자번호가 이미 DB에 존재
**해결**: 기존 업체 삭제 또는 다른 사업자번호 사용

### HTTP 500: 서버 오류
**가능한 원인**:
1. 주소 검색을 하지 않음 (representative_postcode가 null)
2. 신분증 사진을 업로드하지 않음
3. 필수 필드 누락
4. 데이터 형식 오류

**해결**:
1. **주소 검색 버튼 반드시 클릭** ⚠️
2. **신분증 사진 업로드 필수** ⚠️
3. 모든 필수 항목 입력 확인
4. 콘솔 로그 확인 (F12 → Console)
