쿠키 복사 방식

사용 방법 (30초 소요)

  1. 새 탭에서 KnoxHub에 로그인 → 휴대폰 인증번호 입력까지 완료
  2. 로그인 후 F12 눌러 개발자 도구 열기
  3. 상단 메뉴에서 Application 탭 클릭 → 왼쪽에서 Cookies 확장 → zenio5827.cafe24.com 클릭
  4. PHPSESSID 값 더블클릭 → 복사 (Ctrl+C)
  5. 아래 입력창에 붙여넣기 (Ctrl+V) → 버튼 클릭

참고 이미지:

F12 → Application 탭

└─ Cookies

└─ https://zenio5827.cafe24.com

└─ PHPSESSID ← 이 값 복사!

이 값은 저장되지 않으며, 데이터 가져오기에만 일회성으로 사용됩니다.

JSON 업로드 방식

사용 방법 (1분 소요)

  1. KnoxHub에 로그인 후 오토바이 목록 페이지로 이동
  2. F12 → 상단 Console 탭 열기
  3. 아래 스크립트를 복사해서 콘솔에 붙여넣고 Enter
  4. 자동으로 다운로드된 JSON 파일을 아래에서 업로드 
// KnoxHub 데이터 추출 스크립트 v1.0
(function() {
    const data = {
        motorcycles: [],
        contracts: [],
        extracted_at: new Date().toISOString()
    };
    
    console.log('🔍 데이터 추출 시작...');
    
    // 오토바이 데이터 추출 (페이지 구조에 따라 자동 감지)
    const selectors = [
        '.motorcycle-item', '.bike-item', '.vehicle-item',
        'tr[data-motorcycle]', 'tr[data-bike]',
        '.list-item', '.item-row'
    ];
    
    selectors.forEach(selector => {
        document.querySelectorAll(selector).forEach(item => {
            const motorcycle = {
                plate_number: item.querySelector('.plate, .number, [data-plate]')?.textContent?.trim() || '',
                vehicle_name: item.querySelector('.name, .model, [data-model]')?.textContent?.trim() || '',
                chassis_number: item.querySelector('.chassis, .vin, [data-vin]')?.textContent?.trim() || '',
                mileage: parseInt(item.querySelector('.mileage, .km, [data-mileage]')?.textContent?.replace(/[^0-9]/g, '')) || 0,
                model_year: parseInt(item.querySelector('.year, [data-year]')?.textContent?.replace(/[^0-9]/g, '')) || new Date().getFullYear()
            };
            
            if (motorcycle.plate_number || motorcycle.vehicle_name) {
                data.motorcycles.push(motorcycle);
            }
        });
    });
    
    console.log('✅ 오토바이:', data.motorcycles.length + '대 추출');
    
    // JSON 다운로드
    const blob = new Blob([JSON.stringify(data, null, 2)], { type: 'application/json' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'knoxhub_data_' + new Date().toISOString().slice(0,10) + '.json';
    a.click();
    URL.revokeObjectURL(url);
    
    console.log('📥 JSON 파일 다운로드 완료!');
    console.log('📊 추출된 데이터:', data);
    
    return data;
})();
