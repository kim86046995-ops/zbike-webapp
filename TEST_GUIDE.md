#### **Step 2: 대시보드에서 업체 목록 확인**
```

2. "빠른 액세스" 섹션 확인
3. Row 2 > "업체 목록" (보라색 버튼) 클릭
4. 3개 업체 표시 확인:
   ✅ 테스트신규업체 (888-99-11111)

```

#### **Step 3: 새 업체 등록**
```
1. "업체 추가" 버튼 클릭
2. 폼 작성:
   
   업체 기본 정보:
   - 상호명: (주)신규테스트회사
   - 사업자번호: 777-88-99900  ⭐ 새로운 번호 사용!
   
   대표자 정보:
   - 대표자명: 박철수
   - 전화: 010-7777-8888
   - 주민번호: 850505-1234567
   
   대표자 주소:
   - "검색" 버튼 클릭 → 우편번호 검색
   - 예: "강남구" 검색 → 주소 선택
   - 상세주소: 789호

   - 파란 "파일 선택" 버튼 클릭
   - 이미지 파일 선택 (JPG/PNG)
   - 미리보기 확인

3. "등록하기" 버튼 클릭
4. 성공 메시지 확인
5. 자동으로 업체 목록 페이지 이동
6. 방금 등록한 업체 확인 (총 4개로 증가)
```

### 🔥 방법 2: 일반 브라우저 (캐시 삭제 후)

#### **Step 1: 완전 캐시 삭제**
```
Chrome:
1. 설정 (⚙️) → 개인정보 및 보안
2. 인터넷 사용 기록 삭제
3. "캐시된 이미지 및 파일" 체크
4. "전체 기간" 선택
5. "삭제" 클릭

Mac Safari:
1. Safari → 환경설정 → 고급
2. "개발자용 메뉴 막대에서 보기" 체크
3. 개발자 → 캐시 비우기
```

#### **Step 2: 하드 리프레시**
```
Windows: Ctrl+Shift+R 또는 Ctrl+F5
Mac: Cmd+Shift+R
```

## ⚠️ 반드시 확인해야 할 사항

### 1. 필수 입력 항목
```
✅ 상호명
✅ 사업자번호 (위의 새로운 번호 사용!)
✅ 대표자명
✅ 대표자 전화번호
✅ 대표자 주민번호
✅ 대표자 주소 (검색 버튼으로 입력!)
✅ 신분증 사진 (파일 선택으로 업로드!)
```

### 2. 자주 발생하는 오류

#### 오류 1: "대표자 주소를 검색해주세요"
```
원인: 주소 검색 버튼을 클릭하지 않음
해결: 우편번호 옆의 "검색" 버튼 클릭 → 주소 선택
```

#### 오류 2: "신분증 사진을 업로드해주세요"
```
원인: 신분증 파일을 선택하지 않음
해결: 파란 "파일 선택" 버튼 클릭 → 이미지 파일 선택
```

#### 오류 3: "이미 등록된 사업자번호입니다"
```
원인: 이미 사용 중인 사업자번호 입력
해결: 위의 "사용 가능한 새로운 사업자번호" 중 하나 사용
```

#### 오류 4: "서버 오류가 발생했습니다"
```
원인: 네트워크 문제 또는 중복 사업자번호
해결: 
1. 다른 사업자번호로 재시도
2. 시크릿 모드로 다시 시도
3. 캐시 삭제 후 재시도
```
