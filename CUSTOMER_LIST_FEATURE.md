# 계약자 목록 기능 구현

## 📋 개요
계약자 정보를 관리하고 조회할 수 있는 계약자 목록 페이지를 추가했습니다. 사용자가 대시보드에서 계약자 정보를 입력하면 자동으로 계약자 목록으로 이동하며, 입력란은 초기화되어 다른 계약자를 입력할 수 있습니다.

## 🎯 주요 기능

### 1. 계약자 목록 페이지 (/static/customers.html)
- **카드 기반 레이아웃**: 각 계약자를 개별 카드로 표시
- **검색 기능**: 이름, 전화번호, 주소로 실시간 검색
- **상세 정보 모달**: 클릭 시 계약자의 전체 정보를 모달로 표시
- **반응형 디자인**: 모바일, 태블릿, 데스크톱에서 최적화된 레이아웃

### 2. 자동 저장 후 이동
저장 버튼 클릭 → 계약자 정보 저장 → 계약자 목록 페이지로 자동 이동 → 대시보드의 입력란은 자동 초기화

### 3. 계약자 카드 디스플레이
각 카드에 표시되는 정보:
- 이름 (굵은 글씨)
- 등록일자
- 전화번호 (포맷: 010-1234-5678)
- 주민등록번호 (마스킹: 123456-*******)
- 주소
- 상세 정보 보기 버튼

### 4. 상세 정보 모달
- 전체 개인정보 표시
- 주민등록번호 (마스킹 없음)
- 전화번호
- 주소
- 면허 종류 (있는 경우)

### 5. 검색 기능
- 실시간 필터링
- 검색 대상: 이름, 전화번호, 주소
- 대소문자 구분 없음
